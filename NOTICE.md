# Adaptations

The original webpages were adapted from HTML into Markdown for use as a sample
RAG knowledge corpus.

Depending on the source page, the adaptation may include:

- converting HTML to Markdown
- normalizing heading hierarchy
- removing navigation, branding, headers, footers, and page chrome
- removing images, videos, downloads, infographics, and other non-text media
- removing unrelated page elements
- removing nonessential hyperlinks while preserving their visible text
- correcting obvious formatting or typographical issues introduced during
  extraction

Only textual content necessary for retrieval demonstrations has been retained.
Images, branding, navigation, interactive elements, and other nonessential
content have been removed where applicable.

Each document records its original source URL, retrieval date, and adaptation
details in its front matter.

These adapted documents are provided for informational and demonstration
purposes only and are **not** official CDC or NPS publications.

The original agency webpages remain the authoritative source and should be
consulted for the most current information.
